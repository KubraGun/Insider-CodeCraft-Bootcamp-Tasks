Longest Collatz Sequence Finder

The Collatz conjecture is a famous unsolved problem in mathematics. For any positive integer n: if n is even, divide it by 2; if n is odd, multiply it by 3 and add 1. Repeat this process until you reach 1. This app finds which starting number under your input produces the longest chain.

Please enter a valid number between 1 and 1,000,000.
